## 🎬 Recording Tips

### Before Recording:
1. ✅ **Clean Device/Emulator**
   - Clear all previous test data
   - Uninstall and reinstall app for fresh start
   - Set device to portrait orientation
   - Disable notifications
   - Set screen brightness to max

2. ✅ **Prepare Test Data**
   - Have amount values ready (e.g., €45.50, €32.00, €120.00)
   - Prepare descriptions (e.g., "Grocery Shopping", "Bus Ticket", "Movie Night")
   - Plan which categories to use

3. ✅ **Recording Setup**
   - Use screen recording app (AZ Screen Recorder, etc.)
   - Enable "Show touches" in Developer Options
   - Disable battery percentage if distracting
   - Close other apps to prevent lag

### During Recording:
- 🎤 Speak clearly (if adding voice-over)
- 👆 Move slowly and deliberately
- ⏸️ Pause briefly between actions
- 🔄 Show transitions clearly
- ✨ Highlight key features

### After Recording:
- ✅ Trim unnecessary parts
- ✅ Add title screen (optional)
- ✅ Add text overlays for features (optional)
- ✅ Add background music (optional, low volume)
- ✅ Export in HD (1080p minimum)

## 🎯 Key Features to Highlight

### Technical Implementation:
- ✅ **5 Activities** - MainActivity, AddExpenseActivity, ExpenseListActivity, StatisticsActivity, BudgetSettingsActivity
- ✅ **RecyclerView** - Efficient list rendering with ViewHolder pattern
- ✅ **Swipe-to-Delete** - ItemTouchHelper with undo functionality
- ✅ **Data Persistence** - SharedPreferences with Gson serialization
- ✅ **Material Design** - CardView, MaterialButton, TextInputLayout, FAB
- ✅ **Date Picker** - DatePickerDialog for date selection
- ✅ **Input Validation** - Form validation with error messages
- ✅ **Custom Layouts** - Professional UI design

### User Experience:
- ✅ **Intuitive Navigation** - Clear button labels and icons
- ✅ **Visual Feedback** - Toast messages, Snackbars, progress bars
- ✅ **Empty States** - Helpful messages when no data exists
- ✅ **Real-time Updates** - Dashboard refreshes automatically
- ✅ **Color Coding** - Category-based color system
- ✅ **Responsive Design** - Works on different screen sizes

## 📱 USB Device Recording Tips

Since you have your phone connected via USB:

### Enable USB Debugging:
1. Settings → About Phone → Tap "Build Number" 7 times
2. Settings → Developer Options → Enable "USB Debugging"
3. Connect phone to computer
4. Accept debugging prompt on phone

### Screen Recording via ADB (Alternative):
```bash
# Start recording (max 3 minutes by default)
adb shell screenrecord /sdcard/demo.mp4

# Stop recording (Ctrl+C)
# Pull the video to computer
adb pull /sdcard/demo.mp4 C:\Videos\ExpenseTracker_Demo.mp4
```

### Or Use Screen Recording App on Phone:
1. Install "AZ Screen Recorder" or similar
2. Start recording before opening app
3. Perform demonstration
4. Stop recording
5. Transfer video to computer via USB
